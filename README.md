Somewhat simple VU meter using an 8 bit DA/AD PCF8591 via I2C, a LM386 opamp, a 10 LED bar kit, and a sr595 shift register

Parts:
* ICs
    * Raspberry Pi
    * LM386
    * PCF8591
    * SR595 shift register
* Resistors:
    * 10 ohm
    * 22K ohm
    * 10K ohm potentiometer
    * 11 pin 330 ohm resistor network
* Capacitors:
    * 10uF
    * 1uF
    * 47nF
* Misc:
    * electret microphone
    * 10 LED bar package
